# queueTwoStacks.
Implement queue using just two stacks

# Sample Input/Output 

enqueue(value)
Input   	            Arg 	Output
 [1] -> [3] -> X	     5	  [5]->[1]-[3]->X
 
dequeue()

Input   	                Arg 	  Output
 [1] -> [3] -> [4] -> X	     	    4         [1]-[3]->X

## Challenge
Using two stacks implement a queue with enqueue and dequeue features


## Approach & Efficiency
O(N) Linear
